# 🏡 Ez Property

A modern, full-stack property rental and booking platform inspired by Airbnb, tailored for the Nepali market. Built with Laravel, React (Inertia.js), and Tailwind CSS, it provides a seamless experience for both guests and hosts.

## ✨ Key Features

### 👤 For Guests
- 🔍 **Advanced Search & Filters**: Search by location, dates, guests, and property types.
- 📅 **Interactive Calendar**: Real-time availability and dynamic pricing based on custom rules.
- ⭐ **Review & Rating System**: Verified guest reviews with detailed category ratings (Cleanliness, Accuracy, etc.).
- 💳 **Secure Booking**: Transparent price breakdown (Base price, Cleaning fee, Service fee).
- 📱 **Fully Responsive**: Optimized for mobile, tablet, and desktop devices.

### 🏠 For Hosts & Admin
- 🛠️ **Comprehensive Admin Panel**: Manage listings, categories, amenities, and bookings.
- 📸 **Media Management**: Multi-image gallery upload with cover image selection.
- 💰 **Dynamic Pricing**: Set custom prices for specific date ranges (Pricing Rules).
- 📊 **Availability Management**: Block dates, manage bookings, and set minimum night requirements.

---

## 🛠️ Tech Stack

- **Backend**: Laravel 11 (PHP 8.2+)
- **Frontend**: React 18, Inertia.js, Tailwind CSS
- **Database**: MySQL / MariaDB
- **Icons**: Font Awesome 6
- **Tools**: Vite, Axios, Laravel Breeze (Authentication)
