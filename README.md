# 🏡 Ez Property

A modern, full-stack property rental and booking platform inspired by Airbnb and tailored specifically for the Nepali market. Built with Laravel, React, Inertia.js, and Tailwind CSS, Ez Property provides a seamless experience for both guests and hosts.

## ✨ Key Features

### 👤 For Guests

* 🔍 **Advanced Search & Filters**: Search properties by location, dates, number of guests, and property type.
* 📅 **Interactive Calendar**: View real-time availability and dynamic pricing based on custom pricing rules.
* ⭐ **Review & Rating System**: Verified guest reviews with detailed category-based ratings, including cleanliness and accuracy.
* 💳 **Secure Booking**: Transparent pricing breakdown, including base price, cleaning fee, and service fee.
* 📱 **Fully Responsive**: Optimized for mobile, tablet, and desktop devices.

### 🏠 For Hosts & Admins

* 🛠️ **Comprehensive Admin Panel**: Manage properties, categories, amenities, bookings, and other platform content.
* 📸 **Media Management**: Upload multiple property images and select a cover image for each listing.
* 💰 **Dynamic Pricing**: Set custom prices for specific date ranges using flexible pricing rules.
* 📊 **Availability Management**: Block unavailable dates, manage bookings, and set minimum-night requirements.

### ⚙️ System & CMS

* 🌐 **Dynamic Settings**: Manage footer content, contact information, and social media links directly from the Admin Panel.
* 📄 **CMS Pages**: Create and manage dynamic pages such as About Us, Contact Us, Cancellation Policy, and Terms & Conditions.
* ❓ **FAQ Management**: Manage frequently asked questions from the database and display them through an interactive accordion interface.
* 📧 **Multi-Email Contact System**: Send contact form submissions to multiple administrator email addresses simultaneously.

## 🛠️ Tech Stack

* **Backend**: Laravel 11, PHP 8.2+
* **Frontend**: React 18, Inertia.js, Tailwind CSS
* **Database**: MySQL 8.x / MariaDB
* **Icons**: Font Awesome 6
* **Build Tool**: Vite
* **HTTP Client**: Axios
* **Authentication**: Laravel Breeze
